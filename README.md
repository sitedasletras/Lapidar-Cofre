# Lapidar-Cofre
O Lapidar-Cofre é o módulo de ingestão textual estruturada responsável pela leitura diagnóstica inicial da obra, limpeza editorial técnica, separação de elementos estruturais, identificação de metadados autorais e preparação do texto-base para processamento simbiótico pelo Lapidar Pensante e distribuição para os demais módulos do sistema.
